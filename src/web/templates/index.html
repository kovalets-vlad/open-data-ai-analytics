Про проєкт

Цей веб-інтерфейс демонструє результати автоматизованого пайплайну обробки даних (ETL + Machine Learning), розгорнутого за допомогою Docker Compose. Дані завантажуються в PostgreSQL, очищаються, аналізуються на якість, а потім використовується модель RandomForest для прогнозування балансової вартості.
